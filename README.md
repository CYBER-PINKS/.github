# CYBER PINKS

**Cyberpunk-inspired tools, themes, and visual experiments for developers who live in dark mode.**

CYBER PINKS is a small open-source organization focused on neon-flavored developer tools, dark-mode-first themes, and expressive README visuals.

Our visual style is built around deep black surfaces, vivid pink highlights, electric cyan accents, glowing edges, and cyberpunk-inspired contrast.

---

## Featured Project

### Cyber Pink Theme

**Cyber Pink Theme** is a dark Visual Studio Code color theme with neon pink and cyan accents.

It is designed for developers who want a high-contrast cyberpunk atmosphere while keeping the editor readable during long coding sessions.

## Experimental Tools

### Cyber Neon

**Cyber Neon** is an experimental SVG generator for creating animated neon banners, profile visuals, and README assets.

The goal is to make it easy to generate cyber-style SVG visuals from simple text input, without relying on external banner services.

This project is currently in early development and is planned to become part of the CYBER PINKS organization later.

Planned features:

- Text-to-SVG generation
- Neon glow effects
- Flicker, pulse, typing, and glitch-style animations
- GitHub README-friendly SVG output
- Downloadable SVG files
- Copy-ready Markdown snippets

## Design Philosophy

CYBER PINKS is built around a simple idea:

> Developer tools can be practical, readable, and visually expressive at the same time.

We prefer interfaces that feel sharp, dark, electric, and focused.

Our design direction:

- Dark mode first
- High contrast over flat neutrality
- Neon pink as the main brand color
- Cyan as the secondary accent
- Black and deep navy surfaces
- Minimal but expressive motion
- README visuals that feel like part of the project identity

---

## Dark Mode First

CYBER PINKS projects are designed primarily for dark environments.

We care about readability, contrast, and long-session comfort, but our visual identity intentionally favors dark surfaces, neon edges, and cyberpunk-inspired colors.

Light mode compatibility may be considered where appropriate, but it is not the main design target.
